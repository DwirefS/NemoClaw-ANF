# Example Manifests

These example assets show how the enterprise overlay could be deployed without modifying upstream NemoClaw core.

Files in this folder are examples for:

- ANF and Trident backend configuration
- storage classes by ANF tier
- PostgreSQL with `pgvector`
- ingestion triggers
- NIM services
- guardrails
- retrieval API
- worker-tier NemoClaw and OpenShell bootstrap
- private connectivity and secret access

They are deliberately kept in the incubator so they can evolve before any official deployment story is published.
